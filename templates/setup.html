GitHub App Setup

Setup Instructions

To use this GitHub App in a repository, please create a permissions.json file with the following structure:

{
  "users_allowed_to_close_issues": [
    "User1",
    "User2",
    "User..."
  ]
}

The permissions.json file is needed to verify if a user has permission to close or reopen an issue. If the user does not have permission, the app will retrieve a token from the database to perform the action on behalf of an authorized user and execute the opposite operation. It is crucial for authorized users to generate these tokens. You can find the link to generate tokens on the home page of this app.

Place the permissions.json file in the root directory of the repository where the app is installed.
